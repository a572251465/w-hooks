## ✨ 特性

- 易学易用
- 包含大量提炼自业务的高级 Hooks
- 包含丰富的基础 Hooks
- 使用 TypeScript 构建，提供完整的类型定义文件

## 📦 安装

```bash
$ npm install apply-hooks
# or
$ yarn add apply-hooks
# or
$ pnpm install apply-hooks
```

## 🔨 使用

```ts
import { useMount } from 'apply-hooks'
```
